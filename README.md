# FS Tool Pro
![License](https://img.shields.io/badge/license-Apache%202.0-blue)
![Python](https://img.shields.io/badge/python-3.8%2B-blue)
![PyQt5](https://img.shields.io/badge/PyQt5-5.15%2B-orange)

一个轻量级的个人工具，基于 **PyQt5** 构建

---

### 🌟 功能
* 透明时间
* 快捷便签
* 密码生成器
* RSA生成器
* 重命名工具
* 图片工具
* 文件工具
* 创建文件夹
* HASH校验
* 网络工具
* 应用签名
* ...
